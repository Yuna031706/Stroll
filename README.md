# さんぽ道

## 概要
Javaを使用して制作したコンソール上で動作する散歩シミュレーションゲームです。  
様々なイベントに適切な対応をしながら散歩名人になることを目指します。気分（HP）が下がって0になるとゲームオーバーです。  
5日間気分が下がることなく散歩をすることができたら、これまでの散歩中に起きたことに関するクイズに挑戦。  
全問（３問）正解することができたらゲームクリアです。

## 使用技術
Java

## 主な機能
 - プレイヤーの行動選択
 - ランダムイベント
 - 気分（HP）の管理
 - ゲーム進行の分岐

## 実行方法
Mainクラスを実行してください。
